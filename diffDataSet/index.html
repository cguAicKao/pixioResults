最終成效分析

本報告聚焦於訓練完成後的模型表現。Mixed Model 在多數資料集中均展現出優於 Fundus Only 的基準線。

樣本量影響力

資料集依據樣本數 (N) 降序排列。大規模資料集如 APTOS 表現最為穩定，顯示大數據對於深層特徵提取的重要性。

大型資料集表現 (N > 1500)

APTOS2019、MESSIDOR2 與 Glaucoma Fundus 的基準表現極高，遷移學習在這些任務中主要扮演著優化局部特徵與提升穩定性的角色。

中型資料集挑戰 (Retina)

樣本量為 601 筆。此處可觀察到明顯的性能落差，顯示當數據量處於此量級時，影像品質的異質性對模型表現影響最大。

小型資料集效益 (N < 550)

對於 IDRiD 與 PAPILA，導入 ImageNet 權重的 Mixed Model 能顯著改善在資料極度匱乏下的分類邊界問題，實現穩健的準確度。

未來規劃

簡化後的報告更加突出了最終產出的可靠性。後續研究將針對提升比例低於 1% 的資料集進行更細緻的特徵圖 (Saliency Map) 分析。
